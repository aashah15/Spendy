**Currency:** INR (Indian Rupee)

**Why:** The application is designed for Indian users, as evidenced by the ₹ symbol used throughout the UI (expense tracking shows amounts like ₹18,240).

**How to apply:** Use INR currency formatting (₹) for all monetary values, not USD ($).
