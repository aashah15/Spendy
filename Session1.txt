● This is a Flask-based expense tracker web application — appears to be a
  student learning project.

  Structure:
  - Backend: Flask (Python) with routes for landing page, registration, login,
  logout, profile, and expense CRUD operations
  - Database: Custom database module (database/db.py)
  - Frontend: HTML templates with CSS and JavaScript
  - Testing: pytest setup

  Current state:
  - Basic routes are defined but many are placeholders (logout, profile,
  expenses)
  - Comments indicate this is structured as a step-by-step tutorial (e.g.,
  "coming in Step 3", "coming in Step 7")
  - Students are expected to implement the missing functionality
